# Visa Advisory Forms Portal

Electronic immigration forms for the Visa Advisory system, hosted on Azure Static Web Apps.

## Project Structure

```
/
├── index.html                    # Landing page
├── 404.html                      # Error page
├── staticwebapp.config.json      # Azure Static Web Apps configuration
├── css/
│   └── forms.css                 # Shared styles for all forms
├── js/
│   └── form-utils.js             # Shared JavaScript utilities
└── forms/
    ├── inz-1226.html             # Student Visa Declaration
    ├── inz-1012.html             # Student Visa Application (coming soon)
    ├── inz-1014.html             # Financial Undertaking (coming soon)
    └── inz-1200.html             # Additional Information (coming soon)
```

## Setup

### 1. Deploy to Azure Static Web Apps

1. Fork or clone this repository
2. Go to Azure Portal → Create Resource → Static Web App
3. Select Free plan
4. Connect to your GitHub repository
5. Configure:
   - App location: `/`
   - API location: (leave empty)
   - Output location: (leave empty)
6. Click Create

### 2. Configure Power Automate Endpoint

After creating your Power Automate flow with HTTP trigger:

1. Open `js/form-utils.js`
2. Update `FormConfig.submitEndpoint` with your flow URL:

```javascript
const FormConfig = {
    submitEndpoint: 'https://prod-xx.westus.logic.azure.com:443/workflows/...',
    // ...
};
```

3. Commit and push - Azure will auto-deploy

## Form URLs

Forms are accessed with a secure token:

```
https://[your-app].azurestaticapps.net/forms/inz-1226.html?token=[encrypted-token]
```

Tokens are generated by Power Automate when forms are sent to clients.

## Features

- ✅ Responsive design (mobile-friendly)
- ✅ Multi-step wizard with progress indicator
- ✅ Auto-save to browser (every 30 seconds)
- ✅ Client-side validation
- ✅ Conditional logic (show/hide fields)
- ✅ Secure token-based access

## Development

To test locally:

1. Install a simple HTTP server: `npm install -g http-server`
2. Run: `http-server -p 8080`
3. Open: `http://localhost:8080/forms/inz-1226.html?token=test-token-12345`
